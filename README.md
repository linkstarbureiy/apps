This is a small (<20kb) html+.js (+css, for that matter) ad container. I think it's pretty self-explanatory, but still ..enter a YouTube video url to be transmitted (YouTube platform is safe and trusted), put duration (15-60 seconds), and optionally type how many positions you want to 'jump' in the queue. Then select 'rent' (base = 1 cent/second), and voilà!
It can placed on a website (or inside an app that is html-based). It's free, so don't expect me to make substantial changes to it. It works as is, but you will need PayPal api/sdk keys to collect funds (just plug-and-play), AND enable Firebase/Firestore w security rules allowing reads/writes to the 'global_ads' collection. Enjoy ;)
